# lp-analyzer
这是一个在浏览器中即点即用的轻量级运筹学 (OR) 可视化求解器。  在面对复杂的线性规划、整数规划或多维资源调度问题时，仅仅依靠手算和静态图表往往让人抓狂。本项目旨在将枯燥的数学推导转化为直观的“工程仪表盘”。你只需要丢进一串 OCR 识别出来的杂乱公式，它就能自动完成代数标准化、可行域绘制、最值智能捕获，甚至能在连续几何与离散整数点阵之间自由切换。  ✨ 核心黑科技：  动态灵敏度雷达：独创“参数 C 锚点旋转”系统，直观感受价格波动带来的最优解雪崩，自动反推生存区间。  高低维自适应：二维几何作图与多维代数推导（N维松弛/剩余变量替换）无缝切换。  无论你是正在死磕运筹学期末考的学生，还是在进行数模竞赛（MCM/ICM）方案推演的建模手，这套工具都能帮你省下大量画图和试错的时间。
